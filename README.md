FastAPI Task Manager

This is a simple CRUD API built using FastAPI. It allows users to create, read, update, and delete tasks.

Tech Used
- FastAPI
- Python 3.x
- Pydantic

Features
- Create a Task ✅
- Read Task(s) 📖
- Update a Task ✏️
- Delete a Task ❌

1.Install FastAPI and Uvicorn:
